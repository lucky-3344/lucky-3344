## 🚀 快速使用

### 写可研报告
```
1. 打开 [[01_可行性研究框架]]
2. 复制框架到新文档
3. 根据项目填充内容
4. 调用我帮你生成具体章节
```

### 写标书技术应答
```
1. 打开 [[03_标书技术应答框架]]
2. 对照招标文件逐条应答
3. 参考 [[04_标准应答库]] 复用标准描述
4. 调用我帮你润色/优化
```

### 写技术方案
```
1. 打开 [[02_技术方案框架]]
2. 结合 [[04_标准应答库]] 丰富内容
3. 调用我帮你生成技术架构图描述
```
