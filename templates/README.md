## 📖 模板说明

### 01_可行性研究框架（政府投资版）
**依据**：《政府投资项目可行性研究报告编写通用大纲（2023年版）》国家发改委

**章节**：13章
- 项目概况
- 项目建设背景和必要性
- 需求分析与建设方案
- 选址与建设条件
- 工程技术方案
- 环境影响评价
- 节能节水方案
- 网络安全与数据安全
- 项目管理与运营维护
- 投资估算与资金筹措
- 财务分析
- 风险分析
- 结论与建议

---

### 01b_信息化项目可研框架（信息化专用版）
**依据**：工信部《信息化项目可行性研究报告编制指南》

**章节**：13章
- 项目概述
- 必要性和可行性
- 需求分析（含性能需求表）
- 总体设计方案（含架构图）
- 建设内容
- 网络安全与数据安全设计
- 设备/系统配置方案
- 实施计划
- 运行维护保障
- 投资估算
- 经济效益与社会效益
- 风险分析与应对
- 结论与建议

## 🚀 快速使用

### 写政府投资类可研
```
打开 [[01_可行性研究框架]] → 基于2023年通用大纲
```

### 写信息化项目可研
```
打开 [[01b_信息化项目可研框架]] → 基于工信部指南
```

### 写标书技术应答
```
打开 [[03_标书技术应答框架]] → 逐条响应招标文件
参考 [[04_标准应答库]] → 复用标准表述
```
