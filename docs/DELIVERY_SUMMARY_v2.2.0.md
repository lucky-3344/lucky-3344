```python

coord = (116.3974, 39.9093)  # 北京西二旗

# 处理流程
地址 → 逆地理编码 → 关键词分析 → POI查询 → 方向分析
```

## 🚀 使用指南

### 快速开始（5分钟）

#### 1. 配置API密钥

```bash
# 方式A: 环境变量（推荐）
$env:AMAP_API_KEY = "your_api_key"

# 方式B: 配置文件
# 编辑 amap_config.json，替换 api_key 值
```

#### 2. 运行程序

```bash
python google_maps_tool.py
```

#### 3. 单点分析

```
输入坐标: 116.3974, 39.9093
点击"分析"
查看结果: 区域属性 + 地理特征
```

#### 4. 批量处理

```
点击"导入CSV"
选择文件
等待处理
自动生成 results.xlsx
```

### 常见场景

#### 场景1: 物流选址

```python
需求: 找最优快递点位置
结果: 市区 + 路边 + 十字路口 = ⭐⭐⭐
```

#### 场景2: 商业分析

```python
需求: 餐饮店选址评分
结果: 加权计分 (路边权重最高)
```

#### 场景3: 应急管理

```python
需求: 资源部署规划
结果: 高山→防灾资源, 水体→防洪资源
```
