- 🏘️ **乡镇** - 乡镇行政中心
- 🌾 **农村** - 村落地区
- 🏭 **郊区** - 开发区/工业区

## 🎓 技术亮点

### 分类算法

采用**多规则融合**策略：

- **规则1**：POI密度初步分类（主要指标）
- **规则2**：地址关键词微调（精准识别）
- **规则3**：行政区划识别（特殊判断）

### API调用链

```
逆地理编码 → 获取详细地址信息
周边查询 → 统计POI密度
综合分析 → 返回区域属性
```

### 错误处理

- 网络超时捕获（5秒限制）
- 无效坐标格式验证
- API调用失败降级处理

## 🔐 API密钥配置

### 获取方式

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册账号 → 创建应用
3. 申请权限：地理编码、逆编码、周边查询

### 配置方式

```bash

export AMAP_KEY="your_key"

# 或：配置文件
编辑 amap_config.json
```
